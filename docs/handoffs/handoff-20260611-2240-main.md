## Gotchas / 注意点
- **bundle は Phase 0 で1バイトも変えてない**。Phase 1 以降も「成果物は単一HTML」は死守(法則1)
- **テストは `npm test`(要 node_modules、Playwright chromium 導入済み)**。家Macで初回は `npm i && npx playwright install chromium` が要る
- ワーキングコピーは CRLF。node編集は LF正規化→CRLF書き戻し(法則3)
- プレビュー(preview_*)が重いページで劣化したら preview_stop→start でブラウザ再起動(handoff §3)
- 黒塗り出力の検証は「生バイト走査で秘密文字列ゼロ」(偽リダクション検査=血の掟)。テストにも組込済み
- 配布は `git push origin main` で auto-update 発火。未完成を main に push しない
- ps2exe.ps1 は「PDF Compact.exe」を生成するビルドツール。PDF Compact.vbs は静かに起動する現役ランチャー(削除しない)
