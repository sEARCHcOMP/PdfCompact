# PDF Compact

> **Version 3.1.1** · 2026-05-27
> ブラウザ完結型・サーバー不要・100% LOCAL の PDF ツールキット

## ✨ できること

5モードを 1つの HTML に同梱:

| モード | 内容 |
|---|---|
| 🗜 **PDF軽量化** | 書類/写真切替、OCR、4品質プリセット |
| 🖼 **画像→PDF** | JPG / PNG / HEIC / BMP / TIFF / PDF 混在 |
| 🔄 **変換** | 形式変換、個別DL / ZIP |
| ✂️ **PDF編集** | ページ削除 / 並び替え / 回転 / 分割 / 抽出 |
| 📐 **PDFに画像配置** | JWWで作った枠 PDF に写真・図を流し込み |

### v3.0.1 (2026-05-23) — Patch
- プロパティ枠のスクロールバーを常時表示に変更(キャプション追加時に内容が見えない問題を修正)

### v3.0.0 (2026-05-23) — メジャーアップデート: UI再設計
「PDFに画像配置」モードのレイアウト全面刷新。大量画像・複数ページ編集が劇的に快適に:
- 左右ペイン固定 + 中央canvas通常スクロール
- 列幅をドラッグで可変、画面幅追従
- 右ライブラリは画面幅に応じて列数自動増減
- 左サムネに配置画像をプレビュー反映
- ベースPDFページの複数選択 (Ctrl/Shift/ラバーバンド) + 一括削除
- PDFを追加でドロップ → 末尾連結マージ
- ページ削除 × ボタン
- 画像ライブラリ複数選択 (Ctrl/Shift/ラバーバンド)
- 一括ドラッグ配置 / 順次配置モード(🎯)
- 順次配置時にカーソル追従プレビュー(次に置く画像のサムネ + ファイル名)
- 配置画像も複数選択 → ゴミ箱で一括削除

### v2.x — 罫線スナップ & カスタムセル
- 罫線スナップ(JWWの枠線を自動検出)
- 矩形セル中心スナップ + 外枠ハイライト
- カスタムセル(手動矩形 / 編集 / 削除 / 複製)
- 平行アライメントガイド
- Shift+ドラッグで水平/垂直拘束
- 画像配置サイズ 1/3 制限
- 自動アップデート通知 + .exe ランチャー

### v1.x — 基本機能 (Phase 1)
- PDF読込 + ドラッグ&ドロップで画像配置
- 8ハンドルリサイズ / アスペクト比固定
- 整列 / マルチ選択 / スナップ
- 複数キャプション + フォント切替
- `.lpw` プロジェクト保存/読込 + IndexedDB autosave
- PDF出力(ベクター品質保持)

## 🔒 セキュリティ

- **すべての処理はブラウザ内で完結** — ファイルは外部送信されません
- 機密書類でも安心(契約書・施主情報・現場写真など)
- 初回ロード後はオフラインでも動作
- ソースコード公開 — 中身全部見える
