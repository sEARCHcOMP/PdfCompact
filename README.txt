========================================================
  PDF Compact - はじめにお読みください
========================================================

このツールは Web ブラウザだけで動く PDF 整理アプリです。
ファイルは全部あなたの PC 内で処理されます(外部送信ナシ)。


■ インストール手順
----------------------------------------

1. この ZIP を解凍(右クリック → すべて展開)

2. 出てきた「PDF Compact.exe」を、消えない場所に置く
   おすすめ: ドキュメント フォルダの中など

3. PDF Compact.exe をダブルクリックで起動
   ※ 初回はインターネット必要(本体を自動ダウンロード)
   ※ 2回目以降は ネット無しでもOK(オンラインなら自動で最新化)

4. ブラウザが自動で開いてツールが使えます


■ タスクバーに登録する(毎日使うなら)
----------------------------------------

1. PDF Compact.exe を右クリック
2. 「タスクバーにピン留めする」を選ぶ
   ※ Windows 11 の場合: 右クリック → 「その他のオプションを表示」
      → 「タスクバーにピン留めする」

3. 完了。タスクバーから1クリック起動


■ デスクトップにアイコンを置きたい
----------------------------------------

1. PDF Compact.exe を右クリック
2. 「送る」→「デスクトップ (ショートカットを作成)」

3. デスクトップにアイコンが出ます


■ 困ったとき
----------------------------------------

Q: Windows Defender が「不明な発行元」と警告を出す
A: 「詳細情報」→「実行」を押す。
   個人作成の小規模アプリなので署名はナシです。

Q: 起動しても何も起きない
A: インターネット接続を確認。
   またはアンチウイルスが PowerShell をブロックしてる可能性。

Q: 古いバージョンが開かれる
A: 一旦アプリを閉じてもう一度起動すると最新化される。


■ アンインストール
----------------------------------------

PDF Compact.exe と、同じフォルダにある pdf_compact_bundle.html を削除するだけ。
レジストリは一切いじりません。
