PART 01 / PHONE TRANSFER

スマホ転送(スマホとパソコンで送り合う)の使い方

現場で撮った写真を、ケーブルも専用アプリも使わずにやり取り。QRコードを読み取るだけで、スマートフォンとこのパソコンの間で写真やファイルを送り合えます。

STEP-BY-STEP

3ステップで完了します

アプリを開く → スマホで読み取る → 受信する。

01
OPEN

アプリを開く

アプリを開くと最初に「スマホ転送」が表示され(上部タブの一番左)、接続用のQRコードが自動で出ます。この機能だけインターネット接続が必要です(ほかのタブは今までどおりオフラインで使えます)。

02
SCAN

スマホで読み取る

スマートフォンのカメラでQRコードを読み取ると、送信ページが開きます。アプリのインストールは不要です。画面が「スマートフォンと接続中」に変われば準備完了。

03
RECEIVE

受信する

スマートフォンから写真を送ると「受信待ち」に届きます。「受信」を押すとブラウザのダウンロードフォルダに保存されます。まとめて保存するときは「すべて受信」が便利です。

パソコンからも送れます:接続後に「スマートフォンへ送る」の「ファイルを選んで送る」を押す(またはファイルをドラッグ&ドロップ)と、スマートフォン側に「保存」ボタン付きで届きます。受信側は、不要なファイルを✕で見送りできます(保存されず、サーバからも消えます)。
データの扱い:転送したファイルは、受け取り側が保存または見送りした時点で中継サーバから削除されます。タブを閉じると接続そのものが消え、QRコードは開くたびに新しく発行されます。QRコードは接続の鍵なので、画面の写真を共有する際は写り込みにご注意ください。

OCR

OCRって何?

写真の中の文字を、検索できるテキストに変換する機能です。

日本語対応 英数字対応 ブラウザ内で完結

現場写真がズラっと並んだPDFは、普通は文字検索(Ctrl+F)ができません。 OCRを有効にすると、写真に写り込んでいる文字(看板・図面の文字・掲示物など)を自動で読み取り、 あとから検索できるようにします。

BEFORE
画像として保存
検索できない
AFTER · OCR ON
文字として認識
検索できる
初回だけ辞書のダウンロードあり(日本語 約10MB、英語 約4MB)。一度DLすればブラウザに保存されるので2回目以降は早くなります。処理時間は1ページあたり数秒〜数十秒。
縮まないPDFは自動で画像化(v3.9.0〜):スクリーンショット入りWordや白黒スキャンなど、書類モードでは縮まないPDFを「おまかせ(自動)」モードのとき自動で画像化して確実に縮めます。手動で「書類」を選んだ場合はテキスト検索を守るため画像化しません。図面などのベクタPDFも画質を守るため対象外(縮まない理由を画面に表示します)。

PART 04 / CONVERT

フォーマット変換の使い方

JPG・PNG・HEIC・BMP・TIFF・PDFを相互変換。混ぜて投げてもOK、出力はZIPでまとめられます。

01
IMPORT

ファイル投入

変換したいファイルをドロップ。形式ごちゃまぜでOK、何個でもまとめて入れられます。

02
SELECT

出力形式を選ぶ

JPG / PNG / PDF / BMP / TIFF から選択。画質とDL方法(個別 or ZIP)も設定。

03

変換スタート

FORMATS

出力形式の選び方

どの形式が何に向いてるか、ざっくり覚えておくと便利です。

JPG
圧縮・写真向け

写真に最適。ファイルサイズが小さく、メール・LINE・Web共有の定番形式。透過は保持できません。

PNG
無劣化・透過対応

ロゴ・イラスト・スクショ向け。劣化なし、背景透明も保持。ファイルはJPGより大きめ。

PDF
1ファイル1PDF

1枚ずつ個別にPDF化。複数画像を1冊にまとめたい時は画像→PDFモードを使ってください。

BMP
無圧縮・大サイズ

Windows標準の無圧縮形式。劣化ゼロだがファイルは非常に大きくなります。古い業務システム向け。

TIFF
高品質・印刷向け

印刷・スキャン・業務用途で使われる高品質形式。官公庁の書類やFAX系システムで求められることも。

MIXED INPUT

形式ごちゃまぜ投入OK

JPG・PNG・HEIC・BMP・TIFF・PDFを同時に放り込んでも、全部まとめて処理されます。

6形式対応 複数ページTIFF/PDFも展開 全自動変換

たとえばこんな混在でも大丈夫:

入力: 現場写真 JPG × 5 + スクショ PNG × 3 + HEIC × 2 + 仕様書 PDF(10ページ)
出力形式 JPG + ZIP選択 →
出力: 全部JPGに統一された 20枚 を 1つのZIPにまとめてDL
ポイント:複数ページのPDFやTIFFは1ページ=1ファイルとして展開されます。ファイル名は「元の名前_page1」「元の名前_page2」という形式で自動付与。

SPLIT

分割出力の使い方

1つのPDFを複数のPDFに切り分けたい時。分割ポイントを置くだけ。

ページ間の✂チップをクリック ZIPで自動出力 品質劣化ゼロ

例えば20ページの資料を3セクションに分けたい時:

入力: 20ページのPDF
ページ5と12の後ろに✂分割ポイント追加 →
出力:
  part1.pdf (1〜5ページ)
  part2.pdf (6〜12ページ)
  part3.pdf (13〜20ページ)
  → 1つのZIPにまとめてダウンロード
ポイント: 分割ポイントを追加すると、ページの間に青い点線バーと「PART 2 START」タグが表示されて、どこで切れるかが視覚的にハッキリ分かります。
分割時に軽量化トグル

分割ポイントを追加するとファイル名バーに「分割時に軽量化」トグルが出現します。ONにすると分割後の各PDFが自動で軽量化(画像化)されて、ファイルサイズが70〜85%削減されます。

⚠ 軽量化するとテキスト検索ができなくなります。検索性を保ちたい時はOFFのままで。

EXTRACT

選択ページだけ取り出し

必要なページだけ抜き出したい時。ページ右上のチェックボックスで選択→「選択を取り出し」。

書類の必要部分だけ抜粋

50ページの報告書から必要な3ページだけ抜き出して、元請けに送る。元のPDFには手を加えず、抽出したPDFだけダウンロード。

📄 元データはそのまま · 必要な分だけ切り出し
USE CASE B
複数PDFから選り抜き

複数のPDFを投入して、各PDFから必要なページだけ選んで1冊にまとめる。施主提出用の抜粋資料づくりに最適。

📦 複数ソース横断抽出
QUALITY

なぜ劣化ゼロなのか

PDF編集は他モードと違い、ページを画像化しません。

多くのPDF編集ツールは、内部でページを画像に変換してから再構築するため、テキストが画像になってしまう・ファイルサイズが膨らむ・検索性が失われるといった問題が起きます。

PDF Compactの編集モードは、pdf-libのcopyPages()技術を使ってページをオブジェクトとしてそのままコピーしています。元のテキスト、フォント、ベクター線、画像、注釈がオリジナルのまま保持されます。
※しおり(目次)と入力フォームは、複数PDFの結合では引き継がれません。

保持されるもの
  ✓ テキスト(検索・コピー可能)
  ✓ フォント
  ✓ 図面のベクター線(CAD図面もくっきり)
  ✓ 画像の解像度
  ✓ 注釈
  ✓ 回転情報
△ しおり(目次)・入力フォームは、複数PDFの結合では引き継がれません

RULER SNAP

罫線スナップ + カスタムセル

JWWの枠線を自動検出して、画像中心をピタッと枠の真ん中に。

📐 罫線スナップ セル中心XY同時吸付 緑外枠ハイライト 扁平率フィルタ

アクションバーの 📐 罫線スナップ ボタンを押すと、ベースPDFの罫線(直線・矩形)を自動検出して、枠で囲まれたセル中心が 緑の + マーカーで可視化されます。画像をドラッグするとそのセル中心にXY同時にピタッと吸い付き、対象セルが 緑の外枠ハイライト で確認できます。

検出されるセル条件

FAQ

よくある質問

気になるポイントにお答えします。

— 共通 —
ファイルは誰かに送信されますか?

いいえ。すべての処理はあなたのブラウザの中で完結します。ファイルが外部のサーバーに送られることは一切ありません。

そのため、契約書・施主情報・現場写真など、機密性の高い書類でも安心してお使いいただけます。インターネットを切った状態でも(初回のOCR辞書DL後であれば)動作します。

スマホでも使えますか?

はい、スマホのブラウザでも動作します。ただし、パソコンに比べるとメモリが少ないため、小さめのファイル(10MB以下目安)から試してみることをおすすめします。

iPhoneの場合はSafari、Androidの場合はChromeでの動作確認を推奨します。

処理が途中で止まってしまいます

メモリ不足の可能性があります。以下をお試しください:

・他のアプリ・タブを閉じる
・ファイルを1つずつ処理する
・大きなPDFは先に分割、画像は枚数を減らす
・OCRをオフにする(OCRは特にメモリを使います)

それでも改善しない場合は、ブラウザ(Chrome・Edge・Safariなど)を最新版にアップデートしてみてください。

— PDF軽量化について —
どのくらい軽くなりますか?

ファイルの中身によって大きく変わりますが、目安として:

・現場写真のまとめ: 元の1/5〜1/10くらい(例: 50MB → 5〜10MB)
・申請書や契約書: 元の1/2〜1/3くらい
・すでに軽いPDF: ほとんど変わらない場合もあります

変化量は処理後に画面上で確認できます(例: 50.2 MB → 4.8 MB (-90%))。

パスワード付きPDFは処理できますか?

申し訳ありません、パスワード付きPDFは処理できません。先にパスワードを解除してからお使いください(Adobe Acrobatやプレビューアプリなどで解除できます)。

OCRの精度が低いです

OCRは画像の文字を読み取る仕組みなので、写真が鮮明でない・文字が小さすぎる・手書きの場合は精度が落ちます。改善策:

・圧縮レベルを「高画質」に上げる
・認識言語を適切に選ぶ(日本語メインなら「日本語のみ」の方が精度が上がる場合も)
・元のPDFをより鮮明なものに差し替える

— 画像→PDFについて —
対応している形式は?

画像: PNG、JPG、HEIC/HEIF
PDF: 通常のPDF(パスワードなし)

画像とPDFを混ぜて投入OK。たとえば「現場写真10枚+仕様書PDF3ページ」をまとめて1冊のPDFに結合できます。他の画像形式(BMP、TIFF、GIFなど)は現在非対応です。

PDFも入力できるって本当?

はい、PDFも画像と一緒にまとめて投入できます。たとえば:

・現場写真(JPG) + 仕様書(PDF) → 引継書類1冊に
・スクショ(PNG) + 契約書(PDF) → 社内回覧用1冊に
・複数のPDFを結合するだけの使い方もOK

PDFは各ページが画像として取り込まれ、設定した用紙サイズ・画質で出力PDFに結合されます。PDFは1つのブロックとして扱われるので、並び替え時は全ページが一緒に移動します。

HEIC変換が遅い

HEICは一度JPEGに変換してからPDFに入れるため、通常の画像より時間がかかります。1枚あたり数秒が目安で、枚数が多いと数十秒〜数分かかることも。

急ぎの場合は、事前にiPhone側で「写真」→共有→JPEGに変換しておく方法もあります(設定→カメラ→フォーマット→「互換性優先」でJPEGに保存することも可能)。

画像の順番を変えたい

サムネイルをマウスでドラッグ(スマホの場合はタップ長押しでドラッグ)して、好きな位置に移動できます。番号がそのままPDFのページ順になります。

不要な画像は、サムネイル右上の✕ボタンで削除できます(マウスを重ねると表示されます)。

ページの向きがおかしい

画像の縦横比で自動判定しているため、判定が意図と違うことがあります。その場合は:

・画像側を先に回転させてから投入する
・用紙サイズを「原寸(画像サイズ)」にする(比率そのまま)

余白が気になる

配置方法を「用紙を埋める」に変更してください。余白なしで用紙いっぱいに画像が配置されます。ただし画像の端は少しトリミングされます。

書類などトリミングされると困る場合は「全体表示」のまま、用紙サイズを画像の比率に近いもの(例: A4ではなくA3など)に変えるのも手です。

何枚まで入れられますか?

上限はありませんが、50枚を超えると処理が重くなります。スマホでは20〜30枚程度が目安。

大量に処理する場合は、画質を「軽量」にする、もしくは数回に分けて生成してから、別途PDFを結合するのをおすすめします。

ファイル名はどうなりますか?

自動でタイムスタンプ付きの名前が付きます(例: images_20260421_1430.pdf)。年月日と時刻(時分)が入るので、複数作っても上書きされません。

ダウンロード後に好きな名前にリネームして保存してください。

— フォーマット変換について —
形式をごちゃまぜで投げても大丈夫?

はい、全く問題ありません。JPG・PNG・HEIC・BMP・TIFF・PDFを同時に放り込めます。

ただし出力形式は1種類のみです。たとえば「半分はJPG、半分はPDF」のような出力はできません。その場合は2回に分けて実行してください。

複数ページのPDF/TIFFはどうなる?

各ページが別々のファイルとして出力されます。たとえば10ページPDFをJPGに変換すると、元名_page1.jpg〜元名_page10.jpgの10ファイルが生成されます。

逆に1つにまとめたい場合は「画像→PDFモード」を使って結合してください。

ZIPと個別DL、どっちがいい?

ファイル数で使い分けるのがおすすめ:

・3〜5枚以下:個別DLで十分
・それ以上:ZIPの方が速くて整理しやすい

個別DLだとブラウザによっては「複数のファイルDLを許可しますか?」という確認が出ることがあります。ZIPならそれが起きません。

画質設定はいつ効く?

画質設定はJPGとPDF出力時のみ効果があります。

PNG・BMP・TIFFは元から「無劣化」の形式なので、画質を下げてもサイズは変わらず、画質設定は無視されます(UIでも半透明になって無効化が表示されます)。

HEIC→JPG一括変換はできる?

得意分野です!iPhoneの写真(HEIC)を複数ドロップして出力形式「JPG」+「ZIP」にすれば、一気に変換できます。

変換時間は1枚あたり数秒が目安。20枚で1〜2分程度です。

透過(透明背景)は残る?

出力形式によります:

・PNG / TIFF / BMP(32bit):透過を保持
・JPG / PDF:透過は白背景に変換されます

PNGのロゴなどをJPGにすると背景が白くなるので、透過を残したい場合はPNGかTIFF出力を選んでください。

— PDF編集について —
編集すると画質は落ちますか?

劣化ゼロです。PDF編集モードはページを画像化せず、元のPDFオブジェクトをそのままコピーする方式を使っています。

テキスト・フォント・図面のベクター線・埋め込み画像・注釈はオリジナルのまま保持されます。編集後もCtrl+Fで文字検索も可能です。

※しおり(目次)と入力フォームは、複数PDFの結合では引き継がれません。

複数PDFを1つにまとめられる?

はい、簡単にできます。複数のPDFを一度にドロップすると、全ページが並びます。

ドラッグで並び順を整えて「PDFを出力」すれば、1つのPDFとして結合されます。複数PDFを投入した時は自動的に「結合モード」になります。

分割ポイントはどう追加する?

ページとページの間、ページカードの右端に小さな「✂チップ」ボタンがあります。クリックすると青く点灯し、そのページの後ろで分割されます。

分割ポイントが追加されると、ページ間に青い点線バーと「PART 2 START」などのタグが表示され、どこで分割されるかが一目で分かります。

分割と抽出の違いは?

似てますが用途が違います:

分割(✂): すべてのページを使って、境界で区切って複数PDFに。20ページを5,5,10で3分割 → 合計20ページ全部が3つのPDFに入る
抽出(選択を取り出し): 選んだページだけを1つのPDFに。20ページから3ページ選択 → 3ページだけのPDFが出る(残り17ページは使われない)

「選択を右回転」ボタンがグレーアウトしてる

ページが1つも選択されていないためです。ページカードの番号をタップすると選択され、ボタンが有効になります。

複数ページを選んで一括回転もできます。選択中のページは青い枠で表示されます。

パスワード付きPDFも編集できる?

パスワードで保護されたPDFは基本的に編集不可です。先にパスワードを解除してから投入してください。

内部的にはパスワード解除を試みる設定ですが、強い暗号化(AES-256など)が掛かっていると読み込めないことがあります。

「分割時に軽量化」トグルって何?

分割ポイントを置いた時にだけ出現する特別なトグルです。ONにすると、分割後の各PDFが自動で軽量化(画像化)されます。

なぜ必要か: PDFを分割すると、ブラウザ環境の技術的制約で各分割PDFが元のフォント・画像を全部含んでしまうため、解凍すると元サイズ×分割数に膨らむことがあります(例: 5MB→15MB)。

軽量化トグルONで、各PDFが150DPIで画像化されてサイズが70〜85%削減されます。ただしテキスト検索はできなくなる点に注意。

使い分け:
・テキスト検索性を保ちたい → OFFのまま
・ファイルサイズを重視 → ON

ページをチェックボックスで選べる?

はい、各ページの右上にチェックボックスがあります。タップすると青く点灯して選択中になります。

チェックボックスは小さく見えますが、実際のタップ反応範囲は見た目の約4倍に広げてあるので、指でも余裕でタップできます。

選択したページに対して、ツールバーの「選択を取り出し」「選択を削除」「選択を右回転」が有効になります。

— ファイル名について —
ファイル名は指定できる?

はい、全モードで出力ファイル名を自由に指定できます。アクションバーの上にある入力欄に名前を入力するだけ。

空欄のままでもOK。その場合は自動でimages_20260424.pdfのようにタイムスタンプ付きの名前が付きます。

「日時」トグルって何?

ファイル名の末尾に日付(YYYYMMDD)を付けるかどうかのトグルです。デフォルトOFF、押すとON。

例: 入力「見積書」+ トグルON → 見積書_20260424.pdf

同じ名前のファイルを日ごとに作る時に便利。トグルの右には現在の日付プレビューが表示されるので、何が付くかすぐ分かります。

ファイル名に使えない文字は?

Windowsで禁止されている次の文字は自動的に削除されます:

\ / : * ? " < > |

それ以外の文字(日本語、スペース、記号など)は自由に使えます。

変換モードの個別DL時、ファイル名は?

個別DL時は元のファイル名が使われます(拡張子のみ変更)。例えばphoto.heicをJPGに変換するとphoto.jpgで保存されます。

ZIPでまとめる時だけファイル名指定が有効になります。ZIPなら中のファイル名は元のまま保持されつつ、ZIP自体に指定した名前が付きます。
