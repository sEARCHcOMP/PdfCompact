========================================================
  PDF Compact - はじめにお読みください
========================================================

このツールは Web ブラウザだけで動く PDF 整理アプリです。
ファイルは全部あなたの PC 内で処理されます(外部送信ナシ)。


■ インストール手順
----------------------------------------

1. この ZIP を解凍(右クリック → すべて展開)

2. 出てきた「PDF Compact.exe」を、消えない場所に置く
   おすすめ: ドキュメント フォルダの中など

3. PDF Compact.exe をダブルクリックで起動
   ※ 初回はインターネット必要(本体を自動ダウンロード)
   ※ 2回目以降は ネット無しでもOK(オンラインなら自動で最新化)

4. ブラウザが自動で開いてツールが使えます


■ タスクバーに登録する(毎日使うなら)
----------------------------------------

1. PDF Compact.exe を右クリック
2. 「タスクバーにピン留めする」を選ぶ
   ※ Windows 11 の場合: 右クリック → 「その他のオプションを表示」
      → 「タスクバーにピン留めする」

3. 完了。タスクバーから1クリック起動


■ デスクトップにアイコンを置きたい
----------------------------------------

1. PDF Compact.exe を右クリック
2. 「送る」→「デスクトップ (ショートカットを作成)」

3. デスクトップにアイコンが出ます


■ 試してみる用のサンプル (samples フォルダ)
----------------------------------------

解凍したフォルダの samples/ に以下を同梱:
  - 照明器具表枠サンプル.pdf  (空欄の枠が並んだ表)
  - サンプル写真_001.png      (青系のダミー写真)
  - サンプル写真_002.jpg      (暖色系のダミー写真)

「PDFに画像配置」モードで遊ぶ用です:
  1. PDF軽量化等の上にあるタブから「PDFに画像配置」を選ぶ
  2. 照明器具表枠サンプル.pdf をドロップ
  3. 右側ライブラリに サンプル写真_001.png / 002.jpg をドロップ
  4. 写真を枠にドラッグして配置 → PDF出力で完成


■ ウイルス判定・セキュリティ警告が出る時
----------------------------------------

このアプリは個人作成のため、コード署名(有料)を付けていません。
そのため Windows / ブラウザ / アンチウイルスが「未知のアプリ」として
警告を出す場合があります。中身は安全ですので、以下の手順で乗り越え
られます。

▼ ブラウザでダウンロード時に「危険なファイル」と出る
  (Edge / Chrome)
  - 「︙」(三点メニュー)→ 「保持する」を選ぶ
  - Edge は「詳細情報」→「保持する (安全ではない可能性があります)」

▼ 起動時に「Windows によって PC が保護されました」と出る
  - 「詳細情報」をクリック → 「実行」を押す

▼ Windows Defender (ウイルス対策) がブロック
  - スタートメニュー → 「Windows セキュリティ」を開く
  - 「ウイルスと脅威の防止」→「保護の履歴」
  - PDF Compact.exe を見つけて「許可」

▼ 会社の AV が完全ブロック (Trend Micro / Symantec / McAfee 等)
  - IT 管理者に「個人作成の社内ツール、許可リスト登録お願い」と相談
  - もしくは下記の代替起動方法を試す

▼ どうしても .exe が動かない場合の代替手段
  PDF Compact.exe の代わりに「PDF Compact.bat」をダブルクリック。
  黒い画面が一瞬出ますが、同じ動作で起動します。
  AV ソフトに警告されにくい形式です。

▼ ブラウザだけで使いたい (起動 exe 一切使わない)
  解凍したフォルダ内の「pdf_compact_bundle.html」を
  ダブルクリック → ブラウザでそのまま開きます。
  ※ 自動更新はされないので、新版が出たら配信元に再 DL 依頼を


■ 困ったとき
----------------------------------------

Q: 起動しても何も起きない
A: インターネット接続を確認。
   またはアンチウイルスが PowerShell をブロックしてる可能性。
   上記「代替起動方法」も試してください。

Q: 古いバージョンが開かれる
A: 一旦アプリを閉じてもう一度起動すると最新化される。
   それでも古いままなら、ブラウザで Ctrl+Shift+R を押す。

Q: Mac で使いたい
A: pdf_compact_bundle.html をダブルクリックで OK
   (.exe / .vbs / .bat は Windows 専用)


■ アンインストール
----------------------------------------

PDF Compact.exe と、同じフォルダにある pdf_compact_bundle.html を削除するだけ。
レジストリは一切いじりません。
