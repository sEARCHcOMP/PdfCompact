# PDF Compact

> **Version 3.3.9** · 2026-05-28
> ブラウザ完結型・サーバー不要・100% LOCAL の PDF ツールキット

## ✨ できること

5モードを 1つの HTML に同梱:

| モード | 内容 |
|---|---|
| 🗜 **PDF軽量化** | 書類/写真切替、OCR、4品質プリセット |
| 🖼 **画像→PDF** | JPG / PNG / HEIC / BMP / TIFF / PDF 混在 |
| 🔄 **変換** | 形式変換、個別DL / ZIP |
| ✂️ **PDF編集** | ページ削除 / 並び替え / 回転 / 分割 / 抽出 |
| 📐 **PDFに画像配置** | JWWで作った枠 PDF に写真・図を流し込み |

## 🔒 セキュリティ

- **すべての処理はブラウザ内で完結** — ファイルは外部送信されません
- 機密書類でも安心(契約書・施主情報・現場写真など)
- 初回ロード後はオフラインでも動作
- ソースコード公開 — 中身全部見える
